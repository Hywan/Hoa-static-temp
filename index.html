Static and shared resources of Hoa

Here are listed all public static and shared resources of Hoa:

  • Css: UI, layout, HTML and SVG style files;
  • Data: automatically generated data;
  • Font: shared font files;
  • Image: images, avatars, logos etc.;
  • Javascript: javascript files;
  • Map: style, marker, tiles etc.

And here ared listed all external repositories (in External/ directory):

  • D3: D3.js is a JavaScript library for manipulating documents based on data;
  • Prism: Prism is a new lightweight, extensible syntax highlighter, built with modern web standards in mind;
  • Popcorn.js: Mozilla's HTML5 Video Framework;
  • Leaflet: Javascript library for mobile-friendly interactive maps.
